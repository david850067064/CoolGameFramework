安装方法

方法1：通过 Unity Package Manager 安装（推荐）

  1. 打开 Unity 编辑器
  2. 打开 Package Manager（Window > Package Manager）
  3. 点击左上角的 "+" 按钮
  4. 选择 "Add package from git URL..."
  5. 输入以下 URL：
https://github.com/david850067064/CoolGameFramework.git?path=/Assets/CoolGameFramework
  1. 点击 "Add" 按钮

方法2：通过 manifest.json 安装

打开你的 Unity 项目的 Packages/manifest.json 文件，在 dependencies 中添加：

{
  "dependencies": {
    "com.coolgameframework.core": "https://github.com/david850067064/CoolGameFramework.git?path=/Assets/CoolGameFramework"
  }
}

系统要求

  • Unity 2020.3 或更高版本
  • .NET Standard 2.1

使用示例

初始化框架

在场景中创建一个空物体，添加 GameEntry 组件，框架会自动初始化所有模块。

using CoolGameFramework.Core;

// 访问模块
GameEntry.Event.Subscribe<YourEvent>(OnYourEvent);
GameEntry.UI.OpenUI<YourUI>("YourUIName");
GameEntry.Audio.PlayMusic("BackgroundMusic");

事件系统

// 定义事件
public struct PlayerDiedEvent
{
    public int PlayerId;
    public string Reason;
}

// 订阅事件
GameEntry.Event.Subscribe<PlayerDiedEvent>(OnPlayerDied);

// 发布事件
GameEntry.Event.Publish(new PlayerDiedEvent
{
    PlayerId = 1,
    Reason = "Fall"
});

// 取消订阅
GameEntry.Event.Unsubscribe<PlayerDiedEvent>(OnPlayerDied);

UI管理

// 打开UI
var mainUI = GameEntry.UI.OpenUI<MainUI>("MainUI", UILayer.Normal);

// 关闭UI
GameEntry.UI.CloseUI("MainUI");

// 获取UI
var ui = GameEntry.UI.GetUI<MainUI>("MainUI");

资源加载

// 同步加载
var prefab = GameEntry.Resource.Load<GameObject>("Prefabs/Player");

// 异步加载
GameEntry.Resource.LoadAsync<GameObject>("Prefabs/Enemy", (asset) =>
{
    if (asset != null)
    {
        Instantiate(asset);
    }
});

// 实例化
var instance = GameEntry.Resource.Instantiate("Prefabs/Bullet");

对象池

// 创建GameObject对象池
GameEntry.ObjectPool.CreateGameObjectPool("Bullet", bulletPrefab, 10, 100);

// 从对象池获取
GameObject bullet = GameEntry.ObjectPool.Spawn("Bullet", position, rotation);

// 回收到对象池
GameEntry.ObjectPool.Despawn("Bullet", bullet);

Tween动画

// 移动动画
GameEntry.Tween.MoveTo(transform, new Vector3(5, 0, 0), 2f)
    .SetEase(TweenBase.EaseInOutQuad)
    .OnComplete(() => Debug.Log("Move completed!"));

// 缩放动画
GameEntry.Tween.ScaleTo(transform, Vector3.one * 2f, 1f);

// 旋转动画
GameEntry.Tween.RotateTo(transform, new Vector3(0, 180, 0), 1.5f);

// 透明度动画
GameEntry.Tween.FadeTo(canvasGroup, 0f, 1f);

定时器

// 延迟执行
GameEntry.Timer.DelayCall(2f, () => Debug.Log("2 seconds later"));

// 循环执行
var timer = GameEntry.Timer.LoopCall(1f, () => Debug.Log("Every second"));

// 停止定时器
timer.Stop();

音频播放

// 播放背景音乐
GameEntry.Audio.PlayMusic("BGM_Main");

// 播放音效
GameEntry.Audio.PlaySFX("SFX_Shoot");

// 设置音量
GameEntry.Audio.SetMusicVolume(0.8f);
GameEntry.Audio.SetSFXVolume(0.6f);

编辑器工具使用指南

1. AssetBundle 打包工具

打开方式：Unity 菜单栏 → CoolGameFramework → AssetBundle → Build Window

功能说明：

  • 可视化打包 AssetBundle
  • 支持多平台打包（Windows、Android、iOS、WebGL）
  • 自动生成资源清单文件

使用步骤：

  1. 在 Project 窗口中选择要打包的资源
  2. 在 Inspector 窗口底部设置 AssetBundle 名称
  3. 打开 AssetBundle Build Window
  4. 选择目标平台
  5. 点击 "Build AssetBundles" 按钮
  6. 打包完成后，AB包会生成在 Assets/AssetBundles/[Platform] 目录
// 代码中加载 AssetBundle
GameEntry.Resource.LoadAsync<GameObject>("Prefabs/Player", (asset) =>
{
    if (asset != null)
    {
        Instantiate(asset);
    }
});

2. UI 自动绑定工具

打开方式：选中 UI GameObject → 右键菜单 → CoolGameFramework → Add UI Binder

功能说明：

  • 自动绑定 UI 组件到代码
  • 支持命名规则自动识别（如：btn_Start、txt_Title、img_Icon）
  • 减少手动拖拽组件的工作量

命名规则：

  • btn_ 开头 → 自动识别为 Button
  • txt_ 开头 → 自动识别为 Text/TextMeshPro
  • img_ 开头 → 自动识别为 Image
  • input_ 开头 → 自动识别为 InputField
  • slider_ 开头 → 自动识别为 Slider
  • toggle_ 开头 → 自动识别为 Toggle

使用步骤：

  1. 按照命名规则命名 UI 子对象（例如：btn_Start、txt_Title）
  2. 选中 UI 根对象
  3. 右键 → CoolGameFramework → Add UI Binder
  4. UIBinder 组件会自动添加并绑定所有符合规则的组件
  5. 在代码中通过 UIBinder 访问组件
// 在 UI 脚本中使用
public class MainUI : MonoBehaviour
{
    private UIBinder binder;

    void Start()
    {
        binder = GetComponent<UIBinder>();

        // 获取按钮
        Button startBtn = binder.GetButton("btn_Start");
        startBtn.onClick.AddListener(OnStartClick);

        // 获取文本
        Text titleText = binder.GetText("txt_Title");
        titleText.text = "Welcome";
    }
}

3. 场景快速切换工具

打开方式：Unity 菜单栏 → CoolGameFramework → Scene Switcher

功能说明：

  • 快速切换项目中的场景
  • 显示最近使用的场景
  • 支持场景收藏功能

使用方法：

  1. 打开 Scene Switcher 窗口
  2. 窗口会列出 Build Settings 中的所有场景
  3. 点击场景名称即可快速切换
  4. 可以将常用场景添加到收藏夹

4. 资源检查工具

打开方式：Unity 菜单栏 → CoolGameFramework → Resource Checker

功能说明：

  • 查找项目中的丢失脚本（Missing Script）
  • 查找未使用的资源
  • 检查资源引用关系
  • 帮助清理项目，减小包体大小

使用步骤：

  1. 打开 Resource Checker 窗口
  2. 点击 "Find Missing Scripts" 查找丢失的脚本
  3. 点击 "Find Unused Assets" 查找未使用的资源
  4. 查看检查结果，可以批量删除或修复

5. 框架设置窗口

打开方式：Unity 菜单栏 → CoolGameFramework → Framework Settings

功能说明：

  • 配置框架的全局设置
  • 设置资源加载路径
  • 配置日志级别
  • 设置热更新服务器地址

可配置项：

  • 资源加载模式：Resources / AssetBundle
  • 日志级别：Debug / Info / Warning / Error
  • 是否启用文件日志
  • 热更新服务器 URL
  • AssetBundle 输出路径

热更新完整指南

1. 准备工作

安装 HybridCLR（可选）

如果需要热更新 C# 代码，需要先安装 HybridCLR 插件：

  1. 访问 HybridCLR GitHub
  2. 按照官方文档安装 HybridCLR
  3. 配置热更新程序集

2. 打包资源

步骤1：标记资源

  1. 选择需要热更新的资源（Prefab、场景、材质等）
  2. 在 Inspector 底部设置 AssetBundle 名称
  3. 建议按功能模块命名，如：ui/mainmenu、characters/player

步骤2：打包 AssetBundle

  1. 打开 CoolGameFramework → AssetBundle → Build Window
  2. 选择目标平台（Android/iOS/Windows）
  3. 点击 "Build AssetBundles"
  4. 打包文件会生成在 Assets/AssetBundles/[Platform]

步骤3：生成版本信息

创建版本配置文件 version.json：

{
  "Version": "1.0.0",
  "BuildNumber": 1,
  "Files": [
    {
      "Path": "ui/mainmenu",
      "MD5": "abc123...",
      "Size": 102400
    },
    {
      "Path": "characters/player",
      "MD5": "def456...",
      "Size": 204800
    }
  ]
}

3. 部署热更新服务器

服务器目录结构：

hotupdate/
├── version.json          # 版本信息文件
├── Android/              # Android 平台资源
│   ├── ui/
│   │   └── mainmenu
│   └── characters/
│       └── player
├── iOS/                  # iOS 平台资源
│   ├── ui/
│   │   └── mainmenu
│   └── characters/
│       └── player
└── HotUpdate.dll         # 热更新代码（可选）

上传到服务器：

  1. 将打包好的 AssetBundle 文件上传到服务器
  2. 将 version.json 上传到服务器根目录
  3. 确保文件可以通过 HTTP/HTTPS 访问

4. 客户端集成

初始化热更新管理器：

using CoolGameFramework.Core;

public class GameLauncher : MonoBehaviour
{
    void Start()
    {
        // 设置热更新服务器地址
        string versionUrl = "https://your-server.com/hotupdate/version.json";
        string assetUrl = "https://your-server.com/hotupdate/Android";

        GameEntry.HotUpdate.SetRemoteUrl(versionUrl, assetUrl);

        // 检查更新
        CheckUpdate();
    }

    void CheckUpdate()
    {
        GameEntry.HotUpdate.CheckUpdate((success, updateType) =>
        {
            if (success)
            {
                switch (updateType)
                {
                    case UpdateType.None:
                        Debug.Log("已是最新版本");
                        EnterGame();
                        break;

                    case UpdateType.Resource:
                        Debug.Log("发现资源更新");
                        StartUpdate();
                        break;

                    case UpdateType.Optional:
                        Debug.Log("发现可选更新");
                        // 提示用户是否更新
                        ShowUpdateDialog();
                        break;

                    case UpdateType.Force:
                        Debug.Log("发现强制更新");
                        // 必须更新才能继续
                        StartUpdate();
                        break;
                }
            }
            else
            {
                Debug.LogError("检查更新失败");
            }
        });
    }

    void StartUpdate()
    {
        GameEntry.HotUpdate.StartUpdate(
            // 进度回调
            progress =>
            {
                Debug.Log($"下载进度: {progress * 100}%");
                // 更新进度条 UI
            },
            // 完成回调
            success =>
            {
                if (success)
                {
                    Debug.Log("更新完成");
                    // 加载热更新 DLL（如果有）
                    GameEntry.HotUpdate.LoadHotUpdateDLL();
                    EnterGame();
                }
                else
                {
                    Debug.LogError("更新失败");
                }
            }
        );
    }

    void EnterGame()
    {
        // 进入游戏主场景
        GameEntry.Scene.LoadScene("MainScene");
    }
}

5. 热更新 C# 代码（使用 HybridCLR）

步骤1：配置热更新程序集

  1. 创建热更新代码程序集（例如：HotUpdate.dll）
  2. 在 HybridCLR Settings 中添加该程序集
  3. 编译生成 DLL 文件

步骤2：上传 DLL 到服务器

hotupdate/
├── version.json
├── HotUpdate.dll         # 热更新代码
└── Android/

步骤3：客户端加载 DLL

// 下载并加载热更新 DLL
GameEntry.HotUpdate.LoadHotUpdateDLL();

// 或者加载多个 DLL
string[] dllNames = { "HotUpdate.dll", "GameLogic.dll" };
GameEntry.HotUpdate.LoadHotUpdateDLLs(dllNames);

6. 版本管理策略

版本号规则：

  • 主版本号（Major）：重大更新，不兼容旧版本 → 强制更新
  • 次版本号（Minor）：功能更新，兼容旧版本 → 可选更新
  • 修订号（Patch）：Bug 修复 → 资源更新

示例：

  • 1.0.0 → 2.0.0：强制更新（主版本变化）
  • 1.0.0 → 1.1.0：可选更新（次版本变化）
  • 1.0.0 → 1.0.1：资源更新（BuildNumber 变化）

7. 测试热更新

  1. 打包初始版本（v1.0.0）并安装到设备
  2. 修改资源或代码
  3. 重新打包 AssetBundle
  4. 更新 version.json（修改版本号和文件列表）
  5. 上传到服务器
  6. 启动应用，测试热更新流程

8. 注意事项

  • ✅ 确保服务器支持 HTTPS（iOS 要求）
  • ✅ 做好版本回退机制
  • ✅ 热更新前备份本地数据
  • ✅ 提供断点续传功能
  • ✅ 添加 MD5 校验防止文件损坏
  • ⚠️ 不要热更新核心框架代码
  • ⚠️ 测试不同网络环境下的更新流程
